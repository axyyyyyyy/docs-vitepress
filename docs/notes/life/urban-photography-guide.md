# 城市摄影指南：捕捉都市之美

## 1. 城市摄影的基本要素

### 光线运用
- 黄金时间：日出日落前后
- 蓝调时分：黎明黄昏
- 人造光源：霓虹灯光
- 阴影效果：建筑投影

### 构图技巧
- 引导线：道路、铁轨
- 对称美：建筑倒影
- 框架构图：门窗、拱廊
- 重复元素：窗户排列

## 2. 不同场景的拍摄技巧

### 建筑摄影
```
相机设置：
- 光圈：f/8-f/11
- ISO：100-400
- 焦距：24-70mm
- 使用三脚架
```

### 街头人文
```
相机设置：
- 光圈：f/2.8-f/4
- ISO：自动（400-3200）
- 焦距：35mm或50mm
- 快门优先模式
```

### 夜景拍摄
```
相机设置：
- 光圈：f/8-f/16
- ISO：100-800
- 曝光时间：1-30秒
- 必备三脚架
```
